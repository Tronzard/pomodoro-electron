# Tomodo 🍅

A minimal floating Pomodoro timer built with Electron.

## Features

- Adjustable focus and break durations
- Start, pause, resume, and reset
- Draggable floating widget
- Collapsible settings panel
- Always-on-top
- Lightweight and distraction-free

## Controls

- Click the tomato to expand/collapse settings
- Drag anywhere on the widget to move it
- Start/Pause to control the timer
- Reset to return to the initial state
- Use +/- buttons to adjust durations

## Development

Install dependencies:

```bash
npm install
```

Run:

```bash
npm start
```

## Built With

- JavaScript
- Electron
- HTML/CSS
